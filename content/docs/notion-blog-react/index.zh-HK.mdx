## 這是什麼

Notion Blog React 是一個由 Notion 提供支持，使用 React、Next.JS、tailwindcss、TypeScript、Notion-Api-Worker 等構建的博客。

## 長話短說

1. 從 GitHub Fork [**Notion Blog React**](https://github.com/okisdev/Notion-Blog-React) 項目。
2. 改變 [`config/site.config.js`](https://github.com/okisdev/Notion-Blog-React/blob/main/config/site.config.js) 文件裡的內容來更改你的網頁設定。
3. 使用此[連結](https://vercel.com/new/git/external?repository-url=https%3A%2F%2Fgithub.com%2FHarry-Yep%2FNotion-Blog-React)導入至 Vercel。
4. 加入兩個環境變量 `NOTION_API` 及 `NOTION_TABLE_SLUG` 到 Vercel 配置中。
5. 開始部署。

現在你可以通過使用 Vercel 生成的連結來查看你的網頁。
