## 部署

### 本地部署

1. git clone 本仓库

    ```bash
    git clone https://github.com/okisdev/ChatChat.git
    ```

2. 安装依赖

    ```bash
    cd ChatChat
    npm install -g pnpm
    pnpm install
    ```

3. 创建 `.env.local` 文件并填写环境变量

    ```bash
    cp .env.local.example .env.local
    ```

4. 运行

    ```bash
    pnpm run dev
    ```

### 使用 Docker 部署

1. 克隆 Repo

    ```bash
    git clone https://github.com/okisdev/ChatChat.git
    ```

2. 运行以下命令来构建镜像

    ```bash
    docker build -t chatchat .
    docker run -p 3000:3000 chatchat
    ```

或者使用 Docker Hub 上的镜像

```bash
docker run -p 3000:3000 ghcr.io/okisdev/chatchat:latest
```
